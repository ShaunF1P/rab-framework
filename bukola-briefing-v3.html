⚙️ Zero-Cost AI Analysis Engine

We deliberately avoided LLM APIs. Instead, every framework runs a deterministic AI interpretation layer — a score-aware script engine that generates ruthlessly accurate, corporate-grade psychological read-outs in real time based on the exact mathematical output of each user's profile. No API bill. No latency. No hallucinations.

  • Score combinations map to pre-engineered insight scripts tailored per framework
  • Secondary profile detection — if your primary score ties, the engine identifies it
  • All processing is 100% client-side — zero user data leaves the browser unless they email capture

The platform's single biggest architectural upgrade. We went from 9 frameworks to a full 30-framework routing engine across 5 psychometric verticals. This unlocks enterprise use cases in HR, executive coaching, spiritual formation, and couples/relational therapy.

  • 21 new question banks fully authored — all original, clinical-grade questions
  • 21 new calculation functions — domain-appropriate scoring algorithms per framework
  • Full dispatch chain update — calculateResults and renderResults route all 30 frameworks
  • 5-domain UI grid — 6 framework cards per domain with tags, estimated minutes, lock state

A viral sharing engine that transforms assessment results into downloadable, social-media-ready infographics.

Strategic Roadmap

📍 Where We Are Today
Phase 1 (The Engine) is live. 30 frameworks routed. The question banks, scoring, and result renderers are operational. Phase 2 (Monetisation Gate) is the next sprint — the unlock wall, freemium referral loop, and Supabase user accounts.
Phase 1 — ✅ COMPLETE

The 30-Framework Engine

  • 5 Psychometric Domains — Behavioral, Spiritual, Cognitive, Leadership, Relational
  • 30 Frameworks Routed — Full question banks + scoring + AI interpretation
  • Deterministic AI Copilot — Zero-cost, zero-latency insight engine
  • State Persistence — Selected domain/framework survives page refresh
  • GHL + Telegram + Supabase — Full data pipeline operational
Phase 2 — Next Sprint (7–14 Days)

Freemium Monetisation Gate

  • Unlock Gate Modal — 3 free frameworks, then referral/pay/share wall
  • Referral Tracking — Unique referral links tracked per user in Supabase
  • Share-to-Unlock — Share on LinkedIn/X to access premium frameworks
  • Supabase Auth — Anonymous → authenticated user pathway
  • Usage Telemetry — Per-framework completion rates, domain popularity
Phase 3 — 30–60 Days

Enterprise Scale & White-Label

  • White-Label B2B Licensing — Deploy this engine inside a client's domain (churches, coaches, HR firms)
  • PDF Report Generation — Full downloadable diagnostic report per framework
  • Team Assessment Mode — Aggregate team results for leadership/org development
  • Partner Data Portal — Anonymized aggregate behavioral insights for B2B licensing
  • ERP/HRIS Integration — Demo connectors for enterprise systems
